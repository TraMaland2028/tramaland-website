2. TRML 的计算方法

TRML 是一个指标，用来衡量 1,000,000 美元在石油、粮食、金属和能源世界中的真实价值。

基准日期为 2025 年 02 月 15 日。
自该日起，指数持续反映这一百万美元购买力的变化。

如果大宗商品价格上涨，指数系数上升。
这意味着购买同样的资源需要更多美元。
美元相对于现实世界在贬值。

如果大宗商品价格下跌，指数系数下降。
这意味着这一百万美元能够购买更多。

TRML 是一面镜子。
不是市场预期的镜子。
而是现实事物的镜子。
2.1 篮子构成（v1.0 版本）

当前 TRML 篮子反映了全球经济的基础性大宗商品结构。

能源

WTI 原油（CL=F）——20%
天然气（NG=F）——8%

工业与贵金属

黄金（GC=F）——11%
白银（SI=F）——7%
铜（HG=F）——11%
铂金（PL=F）——7%

农业板块

小麦（ZW=F）——9%
玉米（ZC=F）——8%
大豆（ZS=F）——7%
咖啡（KC=F）——4%
糖（SB=F）——4%
棉花（CT=F）——4%

篮子总权重为 100%。

为什么使用带有 =F 的代码

在 TRML 指数中，采用 CL=F、GC=F、ZW=F 等形式的代码。

“=F” 是一种技术性市场代码格式，用于以统一且稳定的方式获取整个商品篮子的行情数据。

这并不意味着该指数是为期货交易者设计的。
这意味着我们为所有篮子成分选择了一个统一、可比的数据来源。

对于部分资产，已采用混合数据方法：
黄金、白银和铂金的计算不仅参考期货数据，也结合现货数据。

这种方式可以降低偏差，使最终指数更加贴近真实的商品篮子价格变动。

总结

“=F” 只是数据格式。

指数的本质，是衡量基础商品篮子价值变化的工具。

在条件允许的情况下，已引入现货数据，以提高结果的准确性与现实相关性。

2.2 计算原则

TRML-base 服务中使用的公式

1. 单个资产的相对价格

r_i = P_i(t) / P_i(0)

其中：
P_i(t) = 当前资产价格
P_i(0) = 基准价格（启动日价格）

2. 期货 + 现货混合（适用于 GC、SI、PL）

首先计算：

r_i_fut = P_i_fut(t) / P_i_fut(0)
r_i_spot = P_i_spot(t) / P_i_spot(0)

随后进行对数混合：

r_i_hyb = exp( (1 - alpha) * ln(r_i_fut)

alpha * ln(r_i_spot) )

其中：
alpha = 0.5

3. 加权几何篮子指数

L = SUM( w_i * ln(r_i) )

若部分数据缺失：

L_norm = L / SUM(w_i_covered)

Index(t) = BaseIndex * exp(L_norm)

BaseIndex = 1,000,000

最低数据覆盖率：85%

4. 篮子系数

K_raw = Index(t) / BaseIndex

5. EMA7 平滑

K_ema7(t) = alpha_ema * K_raw(t)

(1 - alpha_ema) * K_ema7(t - 1)

alpha_ema = 2 / (7 + 1) = 0.25

6. 美元购买力

PP_raw = 1 / K_raw
PP_ema7 = 1 / K_ema7

7. 百分比变化

Delta_K_percent = (K_raw - 1) * 100
Delta_PP_percent = (PP_raw - 1) * 100

8. JSON 输出值（代码实现方式）

TRML_USD_raw = 1,000,000 * K_raw
TRML_USD_ema = 1,000,000 * K_ema7

9. 简单移动平均（SMA）

SMA_n = (1 / n) * SUM_{k=0 to n-1} Index(t - k)

n 属于集合 {7, 30}

10. 日变动限制（抗噪机制）

P_i(t) 属于区间：

[ P_ref * (1 - 0.15) , P_ref * (1 + 0.15) ]

MAX_DAILY_MOVE = 15%

2.3 购买力计算
篮子系数计算

K = Index / BaseIndex

其中：
BaseIndex = 1,000,000

美元购买力

PP_USD = 1 / K

相对基准的购买力变化（%）

Delta_PP_percent = (PP_USD - 1) * 100

示例

如果：

K = 1.10
（篮子上涨 10%）

则：

PP_USD = 1 / 1.10 = 0.909

Delta_PP_percent = (0.909 - 1) * 100
= -9.1%

美元相对于商品篮子走弱。

如果：

K = 0.90

则：

PP_USD = 1 / 0.90 = 1.111

Delta_PP_percent = (1.111 - 1) * 100
= +11.1%

美元相对于商品篮子走强。

K 的上升反映商品篮子价格上涨，以及名义货币相对于基础资源的购买力下降。

2.4 混合报价机制

对于黄金、白银和铂金，采用混合估值机制：

50% — 期货报价
50% — 现货报价

此机制可降低因期货市场特性（如 contango 与 backwardation）带来的潜在偏差。

2.5 数据来源

当前阶段使用公开市场数据源。

所有计算均通过算法自动执行，无人工干预。
